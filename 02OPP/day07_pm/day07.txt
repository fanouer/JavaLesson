面向对象

对象:
   1 代表一个软件业务的个体
   2 对象的本质是一组数据
   
 解决了 "客观事物" 和 "数据" 之间的差别
 
类：（类本质是名称）
   1 是通过对象的抽象，获取公共特征定义类
   2 是对象的结构定义，是对象模板
 
 Java利用类作为模板，成组的分配对象的数据。


public class Emp{
  String name;
  int age;
  char gender;
  double salary;
}
 
Emp e = new Emp();//创建了4个变量
Emp e2 = new Emp();//创建了4个变量

对象的工作原理：

堆：用于分配对象的数据
栈：用于存储方法中的局部变量
 方法中声明的变量，称为局部变量
引用变量：用类声明的变量是引用变量，引用变量的值是
 对象的内存地址值

引用类型变量的特点：
  1 引用变量的值是对象的地址值
  2 两个变量引用同一个对象，两个变量的值是同一个
  对象的地址。
  3 引用类型变量，在逻辑上就是 代词
  4 引用类型可以赋值为null，null表示不引用任何对象
    如果在变量值为null的引用变量上调用属性或方法
    将出现空指针异常。
  如何解决空指针异常：将变量在适当时候赋值，使其引用对象
  就可以避免空指针异常。
 


方法本质是动词
  
   
 Cell c1 = new Cell();
 Cell c3 = c1;//引用类型变量的赋值
 //将c1的值（地址值）赋值给c3，c3 的值与c1
 //具有相同的地址值，c1和c3引用同一个对象
 
 Cell c3 = null;//c3 不引用任何对象
 //如果在变量值为null的引用变量上调用属性或方法
 //将出现空指针异常。
 
 
 Person I = new Person();
 Person you = new Person();
 
 I.love(you);
 
 you = I;
 
 I.love(you);//自恋

 构造器（构造方法）：主要用于初始化对象的属性
  语法: 和类名一样的方法，不能定义返回值

 对象的方法（用于重用算法）
  1 是对象的行为（功能）
  2 是对象的属性相关的计算过程（算法）
  3 对象的方法不要使用 static 关键字！
  4 任何时候需要使用功能，只需要调用方法
	就可以“重用”算法功能！
